# Mini LinkedIn 

## Présentation du Projet
Ce projet consiste à développer l'API backend d'une plateforme de recrutement simplifiée. Elle permet la mise en relation entre des candidats et des recruteurs, sous la supervision d'un administrateur. 

Le système gère la création de profils, la publication d'offres d'emploi, les candidatures et intègre un système d'événements pour le suivi des activités.

---

* **PHP** >= 8.x
* **Composer**
* **Laravel** (Framework principal) 
* **MySQL** ou tout autre SGBD compatible Eloquent 
* **Extension JWT** pour l'authentification
